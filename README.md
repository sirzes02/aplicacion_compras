## Tecnologías principales

- Laravel
- Mysql

## Requisitos para ejecutar

- PHP v7.4 +
- Composer v1.10.15+
- NodeJS V12.19.0+
- Mysql V5.4+

## Proceso de ejecutado

```bash
- git clone https://github.com/sirzes02/aplicacion_compras.git
- cd aplicacion_compras.
- composer install
- npm i
- cp .env.example .env
- php artisan key:generate
- php artisan migrate --seed
```
